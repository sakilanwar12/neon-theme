## 🚀 QUICK START (5 STEPS)

### Step 1: Activate Theme
- Go to **WordPress Admin > Appearance > Themes**
- Click **Activate** on NEON THEME

### Step 2: Create Menu
- Go to **Appearance > Menus**
- Create menu "Main Menu"
- Add menu items (Home, About, Projects, Contact)
- Assign to **Primary Menu** location
- Save

### Step 3: Set Homepage
- Go to **Settings > Reading**
- Select **A static page**
- Create page "Home" if needed
- Set as **Homepage**

### Step 4: Add Projects
- Go to **Projects > Add New**
- Add your portfolio projects
- Set featured image for each
- Publish

### Step 5: Customize (Optional)
- Edit colors in `neon-style.css`
- Change hero text in `front-page.php`
- Add logo in Customization
- Set up contact form (Contact Form 7)

**Done! Your portfolio is live.** 🎉
